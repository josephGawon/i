ILON CHI MANGIGGA SI INPUT

Kaalon hyphen un manggilawaan cha chuwa un man-isu'n vowel.

ka-alon → kaalon
i-it → iit
vfi-ik → vfiik
so-oton → sooton

Ku-an si hyphen chi gawan cha chuwa un mansillavfiali'n vowel chi separadu'n syllable.

kuan → ku-an
kailyan → ka-iliyan
kain → ka-in
teun → te-un

Kaalon hyphen un manggilawaan cha chuwa un consonant, man-isu man winu chipun.

ig-ga → igga
kul-lalapoy → kullalapoy
pis-git → pisgit

Ku-an si hyphen chi gawan cha consonant ya vowel un lakatalin si mansillavfiali'n syllable.

maamamod → maam-amod
apapot → ap-apot
linawa → lin-awa
angangaw → ang-angaw

Kaalon hyphen un mipasali si "I" lu iyindica chi nasionalidad winu ili un lepuwan

I-Britania → Ibritania
I-Israel → Iyisrael
I-Changoy → Ichangoy
I-Tangɏeg → Itangɏeg
I-Vfelekid → Ivfelekid

Ku-an si apostrophe chi manggilawaan cha chuwa'n vowel un separadun chi glottal stop.

lipa-ug → lipa'ug
ista-in→ ista'in
alla-akit → alla'akit

Lu icontract chi "un", isukat si apostrophe chi "u" sakbay iyutup si mamampangu un ugud.

chake un → chake'n
lachongpe un → lachongpe'n

Ngim amma itollong ta:
chipu'n (chipu un[!?])→ chipun
odle'n (odle un[!?]) → odlen

Ya, itollong chi diperensian cha allaya'n ecsampol:
uɏun → head of
uɏu'n → a head that...

CONSONANTS

All consonants in Tagalog are in Lilubwagen. Additionally, these rare consonants can also be found:

Velar ejective affricate

Almost all instances of k in Lilubwagen is pronounced this way. Note the difference in pronunciation:
ked(Lub) [k͡x'ˈɪd] vs kid (Eng)[kˈɪd]

[k͡x']

kuɏang [k͡x'ˈuɥaŋ]

Voiced velar affricate

Compare the two:
Tagalog: mangga [maŋga]
Lilubwagen: mangga [maŋg͡ɣˈa]

[g͡ɣ]

gipan [g͡ɣipˈan]

Voiced labial-palatal approximant

This consonant uses the odd symbol Ɏɏ. This symbol is not always possible to type on a keyboard; I suggest using ÿ when typing using a phone, but not in this dictionary; I haven't put that in the code. The code will automatically convert plain 'y' into 'ɏ' for you.

[ɥ]

aɏed [aɥˈɛd]

Voiceless labiodental affricate

It is best to avoid using 'b' in representing this consonant. [p̪͡f] and [b] are very disparate; the former being phonetically closer to [f]. I'm not about proscribing things, but we have to treat our language with better care. I've found vf most fitting: it is visually distinct(thus much easier to skim over), pf would tend to be pronounced individually(as [p] and [f]). Other people use the symbol Ƀ(and its variants).

[p̪͡f]

vfeɏoy [p̪͡fɪɥˈo̞j]
